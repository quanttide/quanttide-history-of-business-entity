## 实训体系数据标注助研（2016 年起）：帮助科研工具者做数据标注。招的比较少，因为自己的大模型干掉了。任务型，小作业。组员、组长、主管三级，“数字包工头”。

生产实习课程（2024 年底起）：学校直接通过一门课给我们批量输送数据处理实训生。小作业和大作业都有，目前是在尝试分档次。

长期实训/创赛项目组（2023 年起）：公司拿出来项目给同学比赛。项目型，大作业。

定向实训（2025年起）：公司直接选拔人的。

# 正文区公司的高等教育业务有着极其曲折的演化背景。

最早的非商业化活动来源于 2016 和 2017 年厦大课题组时代的内部教学会议。当时每周会有一个固定时间，由成员们分享自己学习的新知识。这种习惯延续至今，形成了公司独特的学习文化。

在 2017 年课题组社团化的一段时间里，我们也积极地在厦大校园内组织了大量的线 Python 编程教学活动。十年前的厦大还没有多少熟悉 Python 的开发者，包括学校的开Python 课老师也不如我们。所以我们的课程受到了来自校友的一致好评。

来自校友圈的共识是，我们在那个年代的厦大定义开辟了 Python 和大数据赛道，把厦大经管带入了大数据时代。最直接的证据是，在我个人毕业和团队走出学校以后，厦大经管甚至整个厦大的大数据积累迅速流于形式，当前的厦大经济宣传大数据和人工智能教学也只能做做样子，拿不出真成果。

（没错，这句话的潜台词是，没有我和公司，厦大经济的数据和算法技术就是个真正的草台班子，但他们心里一点 B 数都没有，还沾沾自喜觉得自己很厉害。）

（我们也没有兴趣和他们 BB，持续发展我们的科研和企业数据处理业务，持续扶持他们的竞争对手提高产出效率，把他们拍死在沙滩上就好了。）
