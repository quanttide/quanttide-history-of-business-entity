# 量潮课堂发展历程

## 课程体系

**非商业化阶段**：2016 年的内部教学开始起步，延续至今。

**商业化阶段**：2017 年底的 Python 基础课程开始起步，延续至今。

## 实训体系

**数据标注助研**（2016 年起）：帮助科研工具者做数据标注，以任务型小作业为主。组织架构为组员、组长、主管三级。目前因自有大模型替代，规模已缩减。

**生产实习课程**（2024 年底起）：学校通过课程批量输送数据处理实训生。包含小作业和大作业，目前正在尝试分档次管理。

**长期实训与创赛项目组**（2023 年起）：公司提供项目供学生参赛或实训，以项目型大作业为主。

**定向实训**（2025 年起）：公司直接选拔人才进行定向培养。

## 历史背景

公司的高等教育业务有着极其曲折的演化背景。

最早的非商业化活动来源于 2016 和 2017 年厦大课题组时代的内部教学会议。当时每周会有一个固定时间，由成员们分享自己学习的新知识。这种习惯延续至今，形成了公司独特的学习文化。

在 2017 年课题组社团化的一段时间里，我们积极地在厦大校园内组织了大量的 Python 编程教学活动。当时厦大校园内熟悉 Python 的开发者较少，我们的课程因此获得了校友的一致好评。

来自校友圈的共识是，我们在这个时期为厦大开辟了 Python 和大数据赛道。但在团队走出学校后，厦大的大数据积累未能持续发展。
